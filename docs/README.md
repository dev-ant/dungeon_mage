# 문서 인덱스

이 문서는 게임 기획 및 구현 문서의 최상위 인덱스입니다.

## 운영 원칙

- 한 문서에는 한 가지 결정 영역만 담습니다.
- 같은 내용을 여러 문서에 반복해서 쓰지 말고 기준 문서를 갱신합니다.
- 섹션 인덱스를 통해 탐색합니다.
- 예전 진입 문서는 삭제 대신 폐기 안내 문서로 유지합니다.
