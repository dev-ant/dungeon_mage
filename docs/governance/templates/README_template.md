## 소스 오브 트루스 규칙

- 최신 기획 또는 운영 기준은 어떤 문서를 우선하는지 적는다.
- 현재 코드/런타임 사실은 어떤 문서를 우선하는지 적는다.
- 상태 추적은 어떤 tracker를 우선하는지 적는다.
- 아카이브는 최신 기준을 덮어쓰지 않는다고 명시한다.

## 수정 규칙

- 무엇을 바꾸려면 어떤 문서를 수정하는지 항목별로 적는다.
- 새 문서를 추가하면 어떤 상위 인덱스를 갱신해야 하는지 적는다.
- 관련 코드가 바뀌었을 때 함께 갱신해야 하는 기준 문서를 적는다.
