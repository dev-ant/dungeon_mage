# 스킬 레벨 규칙

상태: 사용 중  
최종 갱신: 2026-03-27  
섹션: 성장 시스템

## 범위

이 문서는 스킬 경험치, 스킬 레벨, 데미지 계산식, 마스터리 최종 곱 규칙의 기준 문서입니다. 스킬 종합 레벨과 서클 승급 규칙은 별도 문서에서 관리합니다.

## 기본 규칙

- 캐릭터 레벨은 없습니다.
- 모든 스킬은 개별 레벨을 가집니다.
- 스킬 레벨 범위는 1레벨부터 30레벨까지입니다.
- 스킬 경험치는 해당 스킬 또는 해당 속성 마법이 적에게 실제로 가한 피해량만큼 누적됩니다.
- 최종적으로 스킬 위력은 `스킬 자체 계산식`과 `마스터리 최종 배수`를 분리해서 처리합니다.

## 스킬 경험치 규칙

### 개별 스킬 경험치

- 액티브, 설치, 온앤오프, 직접 피해형 버프 스킬은 자신이 준 실제 피해량만큼 자기 자신의 스킬 경험치를 얻습니다.
- 다단히트 스킬은 모든 히트의 실제 피해량을 합산합니다.
- 설치형은 설치물이 준 누적 피해량 전체를 경험치로 환산합니다.

### 속성 마스터리 경험치

- 화염 계열 스킬이 준 실제 피해량은 `파이어 마스터리` 경험치에 누적됩니다.
- 수속성 계열 스킬이 준 실제 피해량은 `워터 마스터리` 경험치에 누적됩니다.
- 빙결 계열 스킬이 준 실제 피해량은 `아이스 마스터리` 경험치에 누적됩니다.
- 번개 계열 스킬이 준 실제 피해량은 `라이트닝 마스터리` 경험치에 누적됩니다.
- 바람 계열 스킬이 준 실제 피해량은 `윈드 마스터리` 경험치에 누적됩니다.
- 대지 계열 스킬이 준 실제 피해량은 `어스 마스터리` 경험치에 누적됩니다.
- 식물 계열 스킬이 준 실제 피해량은 `플랜트 마스터리` 경험치에 누적됩니다.
- 흑마법 계열 스킬이 준 실제 피해량은 `다크 매직 마스터리` 경험치에 누적됩니다.
- 모든 마법 피해는 공통적으로 `매직 마스터리` 경험치에도 누적됩니다.

## 레벨업 기준

### 권장 경험치 공식

- 다음 레벨 필요 경험치:
- `필요 경험치 = 120 + (현재 스킬 레벨 - 1) * 55`

### 누적 감각

- 1레벨에서 10레벨까지는 비교적 빠르게 오릅니다.
- 11레벨부터 20레벨까지는 주력기를 가려내는 구간입니다.
- 21레벨부터 30레벨까지는 장기 숙련 구간입니다.

## 액티브 스킬 데미지 계산식

### 기본 계산식

피해형 스킬의 기본 계산식은 아래를 기준으로 합니다.

`기본 스킬 피해 = (마법 공격력 x 스킬 계수) + 고정 가산치`

여기서:

- `스킬 계수 = 기본 계수 + ((스킬 레벨 - 1) x 레벨당 계수 증가량)`
- `고정 가산치 = 기본 가산치 + ((스킬 레벨 - 1) x 레벨당 가산 증가량)`

### 최종 계산식

`최종 피해 = 기본 스킬 피해 x 속성 마스터리 배수 x 매직 마스터리 배수 x 기타 최종 배수`

## 마스터리 최종 배수 규칙

- 마스터리는 스킬 기본 계수 안에 더하는 방식이 아니라 마지막에 곱하는 방식입니다.
- 따라서 주력 속성에 숙련될수록 후반 효율이 크게 올라갑니다.

### 속성 마스터리 배수 공식

`속성 마스터리 배수 = 1.00 + (마스터리 레벨 - 1) x 0.004`

예시:

- 1레벨: 1.000배
- 10레벨: 1.036배
- 20레벨: 1.076배
- 30레벨: 1.116배

### 매직 마스터리 배수 공식

`매직 마스터리 배수 = 1.00 + (매직 마스터리 레벨 - 1) x 0.003`

예시:

- 1레벨: 1.000배
- 10레벨: 1.027배
- 20레벨: 1.057배
- 30레벨: 1.087배

## 마나 소모와 쿨타임 계산식

### 액티브 스킬 공통 공식

- `최종 마나 소모 = 기본 마나 소모 x (1 - (스킬 레벨 - 1) x 0.006)`
- `최종 쿨타임 = 기본 쿨타임 x (1 - (스킬 레벨 - 1) x 0.0045)`

### 제한 규칙

- 마나 소모 감소는 기본값의 35퍼센트 이하로 내려가지 않습니다.
- 쿨타임 감소는 기본값의 55퍼센트 이하로 내려가지 않습니다.

## 지속시간, 범위, 개체 수 확장 규칙

모든 스킬이 동일하게 확장되는 것은 아닙니다. 해당 스킬이 확장형 태그를 가지고 있을 때만 다음 규칙을 적용합니다.

### 지속시간 증가형

- `최종 지속시간 = 기본 지속시간 x (1 + (스킬 레벨 - 1) x 0.01)`

### 범위 증가형

- `최종 범위 = 기본 범위 x (1 + (스킬 레벨 - 1) x 0.008)`

### 대상 수 증가형

- 기본 목표 수 1 기준으로 10레벨, 20레벨, 30레벨에서 +1

### 투사체 수 증가형

- 8레벨, 16레벨, 24레벨, 30레벨에 추가 투사체 +1

## 마스터리 추가 보너스 구간

마스터리 패시브는 최종 피해 배수 외에도 일정 레벨 구간에서 해당 계열 스킬의 쿨타임과 마나 소모를 추가로 줄입니다.

### 공통 구간 보너스

| 마스터리 레벨 | 효과 |
| --- | --- |
| 5 | 해당 계열 스킬 마나 소모 2% 감소 |
| 10 | 해당 계열 스킬 쿨타임 3% 감소 |
| 15 | 해당 계열 스킬 마나 소모 4% 감소 |
| 20 | 해당 계열 스킬 쿨타임 5% 감소 |
| 25 | 해당 계열 스킬 마나 소모 6% 감소 |
| 30 | 해당 계열 스킬 쿨타임 7% 감소 |

## 설계 의도

- 주력 스킬을 오래 사용하면 체감상 더 부드럽고 강력해져야 합니다.
- 같은 계열 마법을 오래 쓰면 속성 숙련이 누적되어 빌드 정체성이 강해져야 합니다.
- 개별 스킬 성장과 속성 마스터리 성장을 분리해, 사용 패턴과 속성 몰입 모두 보상해야 합니다.
