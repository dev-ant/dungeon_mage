# 서클 승급 규칙

상태: 사용 중  
최종 갱신: 2026-03-27  
섹션: 성장 시스템

## 범위

이 문서는 `스킬 종합 레벨`을 통해 현재 서클을 올리는 규칙의 기준 문서입니다. 캐릭터 경험치 레벨은 존재하지 않으며, 플레이어의 성장 단계는 이 문서의 서클 승급 규칙으로 판정합니다.

## 핵심 원칙

- 캐릭터 레벨은 없습니다.
- 현재 서클은 `스킬 종합 레벨`로 결정합니다.
- 스킬 종합 레벨은 여러 스킬을 폭넓게 육성할수록 오르지만, 주력 스킬 집중 육성도 충분히 보상해야 합니다.
- 서클 상승은 단순 수치 상승이 아니라 `동시 유지 가능한 버프 수`, `고서클 마법 사용 자격`, `장비 해금 조건`에 직접 연결됩니다.

## 스킬 종합 레벨 정의

### 계산 대상

- 액티브 스킬
- 버프 스킬
- 설치 스킬
- 온앤오프 스킬
- 패시브 마스터리 스킬

### 기본 계산식

`스킬 종합 레벨 = 반올림(전체 장착/습득 스킬 레벨 합계 / 계산 대상 스킬 수)`

설명:

- 플레이어가 현재 보유한 모든 성장형 스킬의 레벨을 더합니다.
- 그 값을 계산 대상 스킬 수로 나눕니다.
- 소수점 첫째 자리에서 반올림합니다.

예시:

- 성장형 스킬 20개의 총합 레벨이 118이면
- `118 / 20 = 5.9`
- 스킬 종합 레벨은 `6`입니다.

## 승급 판정 규칙

### 기본 승급표

| 스킬 종합 레벨 | 현재 서클 |
| --- | --- |
| 1 이상 3 미만 | 4서클 |
| 3 이상 6 미만 | 5서클 |
| 6 이상 9 미만 | 6서클 |
| 9 이상 12 미만 | 7서클 |
| 12 이상 15 미만 | 8서클 |
| 15 이상 19 미만 | 9서클 |
| 19 이상 | 10서클 |

### 시작 상태

- 주인공은 서사적으로 이미 `4서클 천재 마법사`입니다.
- 따라서 게임 시작 시 현재 서클은 `4서클`에서 시작합니다.
- 스킬 종합 레벨이 낮아도 시작 서클은 4서클 아래로 내려가지 않습니다.

## 서클별 해금 보상

| 서클 | 주요 해금 |
| --- | --- |
| 4서클 | 버프 동시 유지 2개, 시작 스펠북 운용 |
| 5서클 | 5서클 마법 안정 사용, 장비 옵션 풀 확장 시작 |
| 6서클 | 버프 동시 유지 3개, 중급 조합 버프 빌드 본격화 |
| 7서클 | 고위 설치/광역 스킬 운용 안정화 |
| 8서클 | 버프 동시 유지 4개, 상위 공명 빌드 폭발 구간 |
| 9서클 | 최상위 의식형 마법 준비 단계 |
| 10서클 | 버프 동시 유지 5개, 최종 필살 조합 완성 |

## 편중 육성 보정 규칙

평균만 사용할 경우 다수의 저레벨 스킬을 얕게 올리는 방식이 유리해질 수 있으므로, 주력기 투자도 보상하는 보정 규칙을 둡니다.

### 주력기 보정

- 레벨 20 이상 스킬 1개마다 `스킬 종합 레벨 계산용 보정치 +0.2`
- 레벨 25 이상 스킬 1개마다 추가 `+0.2`
- 레벨 30 스킬 1개마다 추가 `+0.3`

예시:

- 평균 계산 결과가 `7.6`
- 레벨 20 이상 스킬 2개, 레벨 25 이상 스킬 1개가 있으면
- 보정치 `+0.6`
- 최종 판정값은 `8.2`
- 현재 서클은 `8서클`입니다.

## 승급 제한 규칙

서클 승급은 단순 평균만으로 끝나지 않고, 해당 단계에 필요한 최소 숙련 폭을 요구합니다.

### 서클별 최소 조건

| 목표 서클 | 추가 조건 |
| --- | --- |
| 5서클 | 레벨 10 이상 스킬 3개 이상 |
| 6서클 | 레벨 12 이상 스킬 4개 이상 |
| 7서클 | 레벨 15 이상 스킬 5개 이상 |
| 8서클 | 레벨 18 이상 스킬 6개 이상 |
| 9서클 | 레벨 22 이상 스킬 7개 이상 |
| 10서클 | 레벨 26 이상 스킬 8개 이상 |

의도:

- 너무 적은 스킬만 극단적으로 키워 초고서클에 도달하는 것을 방지합니다.
- 다양한 속성과 운용을 익혀야 고서클의 그릇이 된다는 설정과도 맞춥니다.

## 버프 시스템과의 연결

- 4서클일 때 버프 2개를 동시 유지할 수 있습니다.
- 6서클일 때 버프 3개를 동시 유지할 수 있습니다.
- 8서클일 때 버프 4개를 동시 유지할 수 있습니다.
- 10서클일 때 버프 5개를 동시 유지할 수 있습니다.

중간 서클인 5, 7, 9서클은 상위 버프 운용을 준비하는 과도기 구간으로 둡니다.

## 구현 권장 규칙

- 서클은 전투 중 실시간 갱신하지 않고, 휴식 지점 또는 층 이동 시 재계산하는 것을 권장합니다.
- UI에는 `현재 서클`, `다음 서클까지 필요한 종합 레벨`, `추가 조건 부족 항목`을 함께 표시합니다.
- 내부 데이터에서는 `평균값`, `보정치`, `추가 조건 충족 여부`를 분리해 저장합니다.

## 설계 의도

- 플레이어가 여러 마법을 굴릴수록 진짜 엘리트 마법사로 성장하는 감각을 주어야 합니다.
- 동시에 한두 개 주력기만 깊게 파는 스타일도 의미 있어야 합니다.
- 서클 상승이 버프 슬롯 증가와 연결되기 때문에, 승급 순간 전투 감각의 변화가 크게 느껴져야 합니다.
