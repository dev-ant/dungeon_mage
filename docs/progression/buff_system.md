# 버프 시스템

상태: 사용 중  
최종 갱신: 2026-03-27  
섹션: 성장 시스템

## 범위

이 문서는 버프 중심 액션 RPG 감각을 만들기 위한 버프 시스템의 기준 문서입니다.

## 핵심 목표

- 플레이어가 순간적으로 강해지는 뽕맛을 느껴야 합니다.
- 버프는 확실히 강해야 합니다.
- 버프를 어떻게 쌓고 어떤 순서로 켜느냐가 실력과 빌드 정체성을 나누는 핵심이어야 합니다.
- 다만 무한 유지형 안전 장치가 아니라, 긴 쿨타임과 부작용을 동반하는 폭발형 운용이어야 합니다.

## 기본 규칙

- 버프는 중첩 가능합니다.
- 같은 버프 스킬도 중복 시전이 가능합니다.
- 버프는 개별 지속시간을 가집니다.
- 버프의 기본 쿨타임은 긴 편으로 설정합니다.
- 일부 강력한 버프는 종료 후 부작용을 가집니다.

## 동시 유지 가능한 버프 수

버프 동시 유지 한도는 캐릭터 레벨이 아니라 `스킬 종합 레벨`을 통해 올린 서클에 따라 증가합니다.

| 현재 서클 | 동시 유지 가능한 버프 수 |
| --- | --- |
| 4서클 | 2개 |
| 6서클 | 3개 |
| 8서클 | 4개 |
| 10서클 | 5개 |

## 스킬 종합 레벨과 서클 승급

- 개별 스킬 레벨이 쌓이면 전체적인 스킬 종합 레벨도 상승합니다.
- 스킬 종합 레벨이 특정 기준에 도달하면 현재 서클이 상승합니다.
- 서클 상승은 단순 위력 증가뿐 아니라 버프 동시 유지 수 증가라는 매우 체감 큰 보상을 줍니다.
- 정확한 승급 공식과 추가 조건은 [circle_progression.md](/Users/leesanghyun/git-projects/java-projects/old/dungeon_mage/docs/progression/circle_progression.md)에서 관리합니다.

## 버프 중첩 규칙

### 다른 버프끼리의 중첩

- 서로 다른 버프는 기본적으로 함께 적용됩니다.
- 공격, 캐스팅, 이동, 생존, 오라 계열이 동시에 유지될 수 있습니다.

### 같은 버프의 중복 시전

- 같은 버프도 중첩 가능합니다.
- 다만 같은 버프를 여러 번 쌓을 때는 완전 선형 상승보다 점감 구조를 추천합니다.

#### 권장 중복 배율

| 중첩 수 | 적용 효율 |
| --- | --- |
| 1중첩 | 100% |
| 2중첩 | 70% |
| 3중첩 | 50% |
| 4중첩 | 35% |
| 5중첩 | 25% |

예시:

- 한 버프가 공격력 20% 증가라면
- 1중첩: +20%
- 2중첩: +14%
- 3중첩: +10%

## 버프 지속시간과 쿨타임 기준

### 일반 버프

- 기본 지속시간: 8초 ~ 16초
- 기본 쿨타임: 18초 ~ 35초
- 운용 목적: 주기적인 전투 템포 강화

### 강력 버프

- 기본 지속시간: 5초 ~ 10초
- 기본 쿨타임: 35초 ~ 70초
- 운용 목적: 보스, 정예전, 방 정리 순간 폭발

### 필살 버프

- 기본 지속시간: 4초 ~ 8초
- 기본 쿨타임: 60초 이상
- 운용 목적: 극딜 창구
- 종료 후 부작용을 동반할 수 있음

## 부작용 설계 원칙

강한 버프는 전부 무조건 안전하면 안 됩니다. 강할수록 분명한 리스크가 필요합니다.

### 권장 부작용 유형

- 종료 후 마나 고갈 또는 마나 회복 정지
- 종료 후 일정 시간 피해 증가
- 종료 후 이동 속도 감소
- 종료 후 캐스팅 속도 감소
- 종료 후 버프 재시전 금지 디버프
- 유지 중 초당 체력 또는 마나 소모

### 설계 의도

- 버프를 켜는 순간은 통쾌해야 합니다.
- 하지만 사용 타이밍을 잘못 잡으면 역으로 위기 상황을 만들 수 있어야 합니다.
- 즉, 뽕맛과 판단 부담이 같이 와야 합니다.

## 버프 조합 특수효과

특정 조합의 버프를 동시에 사용하면 추가 효과가 발생합니다.

### 조합 설계 원칙

- 단순 수치 합보다 전투 감각이 변하는 효과를 우선합니다.
- 플레이어가 “지금 버프 창을 열었다”는 감각을 분명히 느껴야 합니다.
- 조합은 2개 조합을 기본으로 하고, 일부 고급 조합만 3개 조합으로 둡니다.

### 현재 조합 기준 문서

- 실제 조합 목록, 우선순위, 내부 쿨타임, 발동 효과는 [buff_combo_catalog.md](/Users/leesanghyun/git-projects/java-projects/old/dungeon_mage/docs/progression/buff_combo_catalog.md)에서 관리합니다.
- 이 문서는 조합 설계 원칙과 적용 순서만 관리합니다.

## 전투 감각 목표

- 평소에는 안정적으로 운영하다가
- 버프 2~3개를 맞물리는 순간 전장이 녹아야 합니다.
- 고서클에 도달하면 버프 슬롯이 늘어나면서 “준비된 폭발”의 감각이 더 강해져야 합니다.
- 10서클 근처에서는 5버프 동시 운용으로 사실상 의식에 가까운 폭발 창구를 만들 수 있어야 합니다.

## 구현 권장 규칙

- 버프는 개별 인스턴스로 관리합니다.
- 같은 버프 중첩도 별도 스택 인스턴스로 저장합니다.
- 버프 적용 순서는:
- 기본 스탯 계산
- 개별 스킬 계수 계산
- 일반 버프 가산/배율 적용
- 마스터리 최종 곱 적용
- 조합 특수효과 적용
- 최종 제한치 검사
