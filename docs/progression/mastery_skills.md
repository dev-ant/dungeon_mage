## 공통 적용 규칙

- 마스터리도 일반 스킬처럼 1레벨부터 30레벨까지 성장합니다.
- 직접 시전하는 스킬이 아니라 패시브이므로 개별 피해량이 없습니다.
- 대신 대응 속성 마법이 준 실제 피해량만큼 경험치를 획득합니다.
- 최종 피해 계산에서 마지막 곱연산 단계에 개입합니다.

## 개별 효과 요약

### 매직 마스터리

- 모든 마법의 최종 피해를 강화합니다.
- 모든 마법의 공통 이해도를 높이는 개념입니다.
- 특정 속성 편중 없이도 꾸준히 성장합니다.

### 파이어 마스터리

- 화염 계열 스킬의 최종 피해를 강화합니다.
- 일정 레벨 구간마다 화염 계열 스킬의 마나 소모와 쿨타임을 추가로 줄여 줍니다.

### 아이스 마스터리

## 설계 의도

- 플레이어가 특정 속성에 몰입할수록 빌드 방향이 분명해져야 합니다.
- 마스터리는 스킬 계수에 더하는 방식이 아니라 마지막에 곱하는 방식이기 때문에 후반 성장 체감이 커야 합니다.
- 주력 속성 스킬을 오래 쓸수록 같은 속성군 전체가 강해지는 구조를 만듭니다.
